## Getting Started
You need node.js and mongodb as prerequisits.
Make sure that mongod is running on default port.
Also you have to install grunt-cli and bower:
```
# npm install grunt-cli bower -g
```


After you clone the git:

```
$ npm install
$ bower install
```

Now you can use it:
```
$ grunt
```
